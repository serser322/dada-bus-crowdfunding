# 灰妲 X 三周年 公車募資進度網

## 關於

Relive 一期生 Vtuber[**灰妲**][Dada-youtube-url]，出道三周年啦!!
[灰妲 X 三周年公車募資進度網][Web-link]提供募資情況(已募資完畢)，並可查詢台北、台中、高雄四台公車的最新即時動態，大家一起來追灰妲的公車吧！

## 網站資訊

可查看四台公車動態，包括當下路線、方向、最新停靠站，以及動態地圖連結；也可查看募資相關資訊。
